WordPress Plugins

There are a number of ways of installing useful plugins to your WordPress website. Here's a quick rundown of the most common methods.

Automatic Plugin Installation

Plugins can be added from the admin area of your WordPress website. First, Click the "Plugins" option in the sidebar

Next, select the "Add New" from the drop-down options

You can use the search bar by typing in the name, author name or a keyword of the plugin you want.
